# 创建型设计模式

## [神奇的魔术师--简单工厂模式](./SimpleFactory.js)
> 简单工厂模式(Simple Factory)：又叫静态工厂方法，由一个工厂对象决定创建某一种产品对象类的实例。主要用来创建同一类对象。

## [给我一张名片--工厂方法模式](./FactoryMethod.js)
> 工厂方法模式(Factory Method)：通过对产品类的抽象使其创建业务主要负责用于创建多类产品的实例。
